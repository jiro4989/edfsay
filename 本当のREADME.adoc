= edfsay

edfsayはEDF隊員がセリフを発するコマンドです。
`cowsay` コマンドに影響を受けました。

EDF! EDF!!

== インストール

以下のコマンドでインストールできるはず。

[source,bash]
----
./install.sh   # EDFは仲間を見捨てない。本当だな。
----

== 使い方

[source,bash]
----
$ edfday "EDF! EDF!!"
$ edfday --edf
$ edfday -h
$ edfsay --random
$ edfsay --type chat -n 1 --genre yes,no,scream
$ edfsay -t commlog -n 5 -g scientist
----
