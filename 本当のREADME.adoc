= edfsay

edfsayはEDF隊員がセリフを発するコマンド集です。
`cowsay` コマンドに影響を受けました。

EDF! EDF!!

== インストール

以下のコマンドでインストールできるはず。

[source,bash]
----
./install.sh   # EDFは仲間を見捨てない。本当だな。
----

== 使い方

[source,bash]
----
$ edf say
$ edf say "主砲がきかにゃい！？"
$ edf say --edf
$ edf -h
$ edf words yes angry -n 10
----
